Operator bitvise berlaku untuk tipe data int, long, short, char dan byte
untuk menghitung dari bit ke bit
misalnya dalam var a = 60 dan b= 13
kalau dalam biner akan menjadi sbb
a = 00111100
b = 00001101
perhatikan pada angka 1 dan 0

And
a = 00111100
b = 00001101
a & b = 00001100

OR
a = 00111100
b = 00001101
a | b = 00111101


XOR
a = 00111100
b = 00001101
a ^ b = 00110001


NOT
a = 00111100
~a = 11000011
